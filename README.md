# 💕 Valentine's Day Surprise Website

An interactive, animated Valentine's Day website to surprise your loved one with a creative "Will You Be My Valentine?" proposal. Features an un-clickable "No" button, beautiful animations, and a celebration page!

---


---

## ✨ Features

### 🎯 Question Page (`index.html`)
| Feature | Description |
|---------|-------------|
| 🧸 **Giant Teddy Bear** | Cute animated teddy holding the message card |
| 💕 **Floating Hearts** | Beautiful floating hearts, flowers & butterflies |
| ✨ **Glowing Particles** | Sparkle effects throughout the background |
| 🚫 **Un-clickable No Button** | Runs away from cursor, shrinks, shows funny messages |
| 🎵 **Background Music** | Auto-plays romantic music with toggle control |
| 📱 **Fully Responsive** | Works perfectly on mobile, tablet & desktop |
| 💬 **Funny Messages** | Custom messages when trying to click "No" |

### 🎉 Celebration Page (`celebration.html`)
| Feature | Description |
|---------|-------------|
| 🎊 **Confetti Rain** | Gentle continuous confetti with burst effects |
| 💌 **Love Letter** | Personalized love message card |
| 🏺 **Love Jar** | Tap to reveal "Reasons I Love You" |
| 🤗 **Send a Hug** | Shows custom GIF with heartfelt message |
| 💕 **Love Meter** | Fills with interactions → morphs into ∞ infinity |
| ⏰ **Countdown Timer** | Live countdown to Valentine's Day |
| ✨ **Sparkle Cursor** | Sparkles follow your mouse movement |
| 🎵 **Background Music** | Toggle on/off with visualizer button |
| 🌧️ **Emoji Rain** | Hearts & flowers continuously falling |

### ⌨️ Secret Keyboard Shortcuts
| Key | Action |
|-----|--------|
| `L` | Love confetti burst |
| `H` | Send virtual hug |
| `K` | Special hearts explosion |
| `I` | Instant infinity mode |
| `J` | Open love jar |
| `C` | More confetti |
| `G` | GIF hug animation |
